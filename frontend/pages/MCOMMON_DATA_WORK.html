PL/SQL 바인드 변수 규칙

왼쪽 Parameter List의 Parameter 명칭은 PL/SQL 스크립트에서 바인드 변수 또는 문자열 치환 키로 사용할 수 있습니다.

  • 바인드 변수: Parameter 명칭이 ABC_DEF이면 스크립트에서는 :abcDef로 작성합니다. 언더바는 제거되고, 언더바 다음 첫 글자만 대문자로 변환됩니다.
  • 동적 문자열 치환: /* --DYNAMIC_MODEL_NAME-- */처럼 작성하면 /* --와 -- */ 사이의 이름이 Parameter 명칭과 완전히 같을 때 해당 값이 문자열 그대로 치환됩니다.
  • Generate PL/SQL: 기본값이 있으면 기본값을 스크립트에 직접 채우고, 기본값이 없으면 :abcDef 형식의 바인드 변수로 생성합니다.
  • Run now / Queue Batch: 실행 시 저장된 Parameter List 값으로 바인드 변수와 동적 문자열 치환을 처리한 뒤 실행합니다.
