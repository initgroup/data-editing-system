How to use
  1. 1. Select model family Choose the reusable model family and review its training source and consumer.
  2. 2. Review confirmed labels Use Training Data to check only user-confirmed labels, duplicates, and conflicts.
  3. 3. Train a candidate Set bounded training options and create a candidate without replacing the active model.
  4. 4. Review validation summary The same training run records holdout, class, group, and confusion metrics for comparison.
  5. 5. Compare and activate Validate the candidate, then activate it only after comparison. Rollback remains available.
