HOME 도움말

INIT Data Editing System의 HOME 화면은 로그인 후 현재 작업 상태, Target DB 연결 상태, 워크플로우 준비도, 공지사항, 주요 메뉴 바로가기를 확인하는 대시보드입니다.

메뉴 요약

사용자가 HOME 화면에서 질문하면 이 문서를 먼저 기준으로 삼습니다. HOME은 작업을 직접 편집하는 화면이라기보다 현재 사용자 세션과 데이터 편집 작업 준비 상태를 확인하고, 다음 업무 화면으로 이동하는 시작점입니다.

화면 목적

  • 현재 로그인 사용자, 권한, Target DB 선택 상태를 확인합니다.
  • 통합 플로우와 데이터 품질 관련 주요 지표를 요약합니다.
  • 공지사항과 작업 바로가기를 통해 사용자가 다음 작업으로 빠르게 이동하게 합니다.
  • 현재 메뉴가 HOME임을 설명하는 도움말 기준 문서로 사용됩니다.

주요 영역

상단 요약 영역
시스템 이름, 사용자 인사말, 로그인/권한/Target DB 상태를 보여줍니다.
Workflow readiness
통합 플로우 실행 준비도와 최근 실행 흐름을 요약합니다. 새로고침 아이콘으로 최신 상태를 다시 조회합니다.
Quality signal / Notices
데이터 품질 알림과 공지사항을 보여줍니다. 공지사항 목록 버튼을 누르면 상세 목록을 확인할 수 있습니다.
Main shortcuts
프로젝트, 시나리오, 데이터 작업, 플로우 작업 등 자주 사용하는 메뉴로 이동하는 바로가기입니다.

헤더 버튼

  • Refresh current page: 현재 열린 화면을 다시 로드합니다.
  • Close current page: 현재 화면을 닫습니다.
  • Close all pages: 열려 있는 업무 화면을 모두 닫습니다.
  • Logout: 세션 정리 후 로그아웃합니다.
  • HOME Help: 이 도움말 팝업을 엽니다.

질문 참고 자료 기준

  1. HOME 관련 질문이면 이 도움말을 가장 먼저 요약합니다.
  2. 현재 화면의 Elements DOM 스냅샷이 첨부되면 실제 렌더링 상태를 기준으로 답변합니다.
  3. 사용자가 캡처 이미지나 파일을 첨부하면 해당 첨부를 질문의 직접 근거로 사용합니다.
  4. 도움말과 화면 상태가 다르면 현재 렌더링된 화면 상태를 함께 언급하고, 확인이 필요한 부분을 분리해서 말합니다.
