목표:
현재 활성화된 파일의 조회 버튼 클릭 로직만 점검하고 수정해줘.

범위 제한:
- 전체 프로젝트를 스캔하지 마.
- 현재 열려진 파일안에서만 확인해.
- 다른 파일은 내가 요청하기 전까지 열거나 수정하지 마.
- 공통 유틸, 라우터, 전역 설정 파일은 추측하지 말고 필요하면 먼저 질문해.

수정 조건:
- 기존 화면 구조와 변수명은 최대한 유지해.
- 불필요한 리팩토링은 하지 마.
- 변경 전/후 차이를 간단히 설명해.

완료 기준:
- 수정한 파일 목록을 알려줘.
- 실제 변경한 코드만 보여줘.
- 추가로 확인이 필요한 부분은 별도 항목으로 알려줘.
